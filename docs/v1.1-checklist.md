# MyPalette v1.1 Completion Checklist

## Goal

MyPalette v1.1은 사용자가 카테고리별로 기록한 모든 컬러를
Pieces of Me 페이지에서 한 화면에 모아볼 수 있도록 한다.

v1.1의 목표는 스티커 UI를 완성하는 것이 아니라,
여러 카테고리에 흩어진 데이터를 하나의 화면에 통합하고
기존 상세 Modal을 재사용하는 기본 사용자 흐름을 완성하는 것이다.

---

## Required for v1.1

### Feature

- [x] Pieces of Me 메뉴로 이동할 수 있다.
- [x] Pieces of Me 전용 페이지가 표시된다.
- [x] 모든 카테고리의 SubColor가 한 화면에 표시된다.

## Done Criteria

아래 조건을 만족하면 MyPalette v1.1을 완료한 것으로 본다.

- 사용자가 Pieces of Me 페이지에 접근할 수 있다.
- 모든 카테고리의 SubColor가 한 화면에 표시된다.
- 각 SubColor가 원래 카테고리 정보를 유지한다.
- 컬러 카드 클릭 시 기존 상세 Modal을 통해 정보를 확인할 수 있다.
- 기존 MyPalette와 ColorGroupDetail 기능이 깨지지 않는다.
- README와 관련 문서에 v1.1 변경 사항이 반영되어 있다.
- `v1.1.0` tag와 GitHub Release가 생성되어 있다.
